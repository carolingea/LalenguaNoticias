Spotlight your Posts using Focus - a multi purpose WordPress Featured Slider Widgets and Shortcode Plugin to display your posts elegantly.

== Description ==

<h3>Featured Posts Content via Shortcode</h3>
The easiest way to add Featured Posts Slider on every WordPress site using shortcodes and/or widgets. <strong>Focus</strong> gives you the simplest way to add and edit featured contents shortcode via media WordPress frame.

[vimeo https://vimeo.com/178769345]

<h3>Featured Posts Slider Widget</h3>
Smoothly integrated on WordPress Widgets to provide you excellent option to add Featured Posts Widget Slider on any of your sidebar contents. Powered by tabbed-options for easiest accessible options to change slider settings.

<h3>Customizer Ready WordPress Featured Posts Plugin</h3>
Beginner ready! You can add the featured posts widget directly via customizer to see the changes live!

<h3>Features</h3>

* Simple and Easiest to user Interface
* Add Responsive WordPress Featured Posts Slider via Shortcode
* Featured Posts Slider Widget
* Fully Responsive
* Fast and Lightweight Slider
* Featured Image Support
* Mobile Touch Navigation Support

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Go to Appearance > Widgets, you'll now have Focus: Featured Slider widget
4. I hope you'll love this plugin :)
